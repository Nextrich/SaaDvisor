# SaaDvisor
Проект создан командой "IP", 
## Stack
- Python (FastAPI, Jango)
- HTML
- CSS
- JS

## Быстрый старт
В папке с фронтом:
```
python -m http.server 3000

```
В папке  с бэком:
```
uvicorn app.main:app --reload --port 8000
```
